## Projects

### 📞 Telco Customer Churn Analysis
**Tools:** Python · Pandas · Scikit-learn · Seaborn  
**Dataset:** 7,043 telecom customers · 21 features

- Cleaned and encoded the full dataset (handled missing values, label encoding, one-hot encoding)
- EDA revealed month-to-month customers churn at ~43% vs ~3% for two-year contracts
- Built and compared Logistic Regression vs Random Forest — RF achieved **AUC = 0.83**
- Simulated two retention strategies; identified **$139K+/month** in revenue at risk
- Delivered an executive memo with priority customer segments and action plan
